# BEA ACF Taxonomy Term 1.0.0

Use library select2 to allow selection of multiple taxonomies and terms as ACF field

## Compatibility

Compatible with :
 - ACF v4
 - ACF PRO : https://www.advancedcustomfields.com/pro/

## Features

- multiple selection, autocomplete
- single selection
